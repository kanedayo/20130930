# Ruby初心者勉強会in沖縄 2013/09/24

## 今日の課題

- githubを使いこなせるようになろう！！

### 勉強の進め方

1. githubへのアカウント登録
2. gravator
3. リポジトリ制作
4. commit
5. push
6. 複数人との共同編集に挑戦
7. forkする
8. forkしたリポジトリをcloneする
9. 書き換えて…
10. commit〜pushする
11. pull requestする
